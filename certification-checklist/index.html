Secure Funder™ Certification Checklist

Independent verification of data protection standards

1 Email Security
Multi-factor authentication enabled
Submission handling: Automated processing OR email gateway watermarking
Outbound email monitoring rule configured for back-office staff
2 Portal Access
Screenshot protection OR screen watermarking if SSN accessible
File access: View-only OR watermarked downloads
3 CRM Security
Screenshot protection OR screen watermarking for CRM users
Document storage: View-only OR watermarked downloads
Bulk export restricted OR export logs maintained
4 Device & Network
Printers disabled OR print watermarking enabled
5 Third-Party Access
BPO/contractor access: View-only OR watermarked documents
6 Cloud Storage
View-only access OR watermarked downloads (if using cloud storage)
7 Personnel Controls
Background checks appropriate for roles
Documented off-boarding process for system access revocation
